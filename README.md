# 🎬 Shorts & Reels Automator (Vertical Video Stacker)

A lightweight, powerful Streamlit web application that seamlessly stacks two landscape videos into a single, viral-ready vertical (9:16) format. Perfect for creating YouTube Shorts, Instagram Reels, and TikToks without opening heavy editing software.

## ✨ Features

* **Smart Resizing:** Choose between `Crop` (fills screen, no black bars), `Fit` (keeps original aspect ratio with black background), or `Stretch`.
* **Flexible Duration:** Auto-sync to the shortest video, loop the shorter video to match the longer one, or set a custom manual duration.
* **Audio Mixer:** Keep top audio, bottom audio, mix both together, or mute everything. Includes a master volume slider.
* **Precision Trimming:** Set precise start times (offsets) for both top and bottom videos to skip intros.
* **Rapid Preview:** Generate a 5-second preview instantly before committing to a full, high-quality render.
* **Cloud Ready:** Fully optimized to run on Streamlit Community Cloud.
